First steps
1. vagrant box add centos/7
2. vagrant plugin add vagrant-vbguest

Basic
1. cd basic
2. vagrant up
3. vagrant ssh
4. cd /vagrant
5. ls -alh
6. exit
7. vagrant destroy -f

Port forward
1. cd port-forward
2. vagrant up
3. Open web browser to http://http://127.0.0.1:8000
4. vagrant destroy
